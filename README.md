# LocalNote(ローカルPDF要約アプリ)

## 概要
　このアプリはPDFをアップロードすることでAIが自動で要点整理や質問に答えてくれます. NotebookLMに似ていますが, このアプリは完全ローカルなためConfidentialな情報も安心して扱えます.

## 動機
　このアプリを制作した理由は, LLMを使って何か作りたかったからと, 大学の講義資料などのあまり外部に出さないほうがいいファイルを扱える要点整理アプリが欲しかったからです.

## ⛏️使用技術
- **フロントエンド**: Streamlit
- **バックエンド**: Python, LangChain
- **ローカルLLM**: Gemma4

>**LangChainとは**:<br>
　LangChainはLLMのモデルを実用的なアプリに組み込むためのツールで, LLM専用の統合管理フレームワークです.

## ⚙️機能・特徴
### 機能
- PDFを読み込んでRAG(*Retrieval-Augmented Generation*)を使用したチャット作成します.
- AIがアドバイザーとなり, 要点整理や問題作成などを行います.

### 特徴
- 1つだけでなく, 複数のチャットの作成が可能です.
  
- 事前にLLMに読ませるプロンプトを設定できます(「書かれてある情報のみを使って」など).

- Gemma4だけでなく, いろいろなLLMモデルを使えます(モデルのダウンロードが必要です).

## 🧠工夫した点
1. **会話の保存・考慮**<br>
   　生成AIにプロンプトを投げる際, 毎度コンテキスト内学習用を行ってから新しいプロンプトのみを読み込みます. そのため, 過去の会話履歴が考慮されない課題があります. 下記の会話がその例です.
   ```text
   User: ○○について詳しく説明して
    AI : ~~です。

   User: それのメリットは？
    AI : ご質問がどの分野のメリットか特定できませんでした。
   ```
   　この課題を解決するために**会話の保存**を行いました. もっと具体的な方法を述べると, **これまでの会話の履歴を保存して, 次の会話の時に履歴を基に新しいプロンプトをAIが理解できるようにAIで変換してから読み込んでいます**.
   ```text
   User: ○○について詳しく説明して
    AI : ~~です。

   User: それのメリットは？ → 「それ」が何か予測して「○○のメリットは？」に変換
    AI : ~~というメリットがあります。
   ```

2. **モデルの自由度**  
   　このアプリの開発に使用したLLMのモデルは*Gemma4:e4b*で, ローカルLLMの中ではミドルクラスの性能です. しかし, このモデルの推奨環境に合っていないので回答に時間がかかってしまったり, より高性能なモデルを使いたい場合が想定されるため, **設定画面で環境にあった好みのモデルを選択できるようになっています**.   
   　初期値では*Gemma4:e4b*より軽いモデルである*Gemma4:e2b*が選べるようになっています. その他のモデルに関しては「その他」を選択してモデル名を入力してください.
   > *Gemma4:e4b*だけでなく, 使用したいモデルは全てダウンロードが必要です.

## 📥インストールについて
### 制作環境
- Python 3.11(64bit)
- Ollama 0.20.2

### インストール方法
1. **レポジトリのデータをダウンロード**
   ```bash
   git clone https://github.com/kiyo1204/LocalNote
   ```
2. **必要なモジュールのインストール**
   ```bash
   pip install -r requirements.txt
   ```
3. **ローカルLLMのダウンロード**
   1. **Ollamaをダウンロード**  
      ダウンロードは以下のページから行います.  
      (https://ollama.com/download/windows)  
      ダウンロードが完了したら, ターミナルで以下のコマンドでダウンロードされているか確認します.
      ```bash
      ollama --version
      ```
   2. **LLMのダウンロード**  
      LLMは初回は*Gemma4:e4b*をダウンロードします(他のモデルは動作確認後に導入してください).  
      ターミナルで以下のコマンドを入力してください.
      ```bash
      ollama pull gemma4:e4b
      ```

4. **起動**  
   *Start.bat*をダブルクリックで実行してブラウザに表示されれば成功です.


## 🖱️使い方
1. **起動**  
   *Start.bat*をダブルクリックで起動します.
2. **チャットの作成**  
   サイドバーの**新規チャットの作成**を選択し, チャット名とファイル(PDF)をアップロードしてから**作成して実行**ボタンをクリックします.
3. **質問**  
   アップロードした資料を基にAIが回答します. 事前に読み込ませるプロンプトを変更する際は設定画面から変更できます.
